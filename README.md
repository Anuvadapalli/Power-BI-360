# 📊 Business Insights 360 - Power BI Project

## 📌 Project Overview
This project provides a comprehensive Business Insights dashboard built using Power BI. It helps stakeholders monitor performance across multiple business functions and make data-driven decisions.

---

## 🎯 Objectives
- Analyze financial performance
- Track sales trends
- Understand marketing effectiveness
- Monitor supply chain efficiency
- Provide executive-level insights

---

## 🛠 Tools & Technologies
- Power BI
- Excel / CSV Data Sources
- DAX (Data Analysis Expressions)
